# Math 3480 Course Schedule - Spring 2024
These are the topics that we plan to cover this semester:
* Exploratory Data Analysis
* Types of Machine Learning
* Data Validation
* Models
  * Linear Regression

* Natural Language Processing

* Generative AI
  * ChatGPT
  * Ethics
